## Se connecter avec un compte local

- **Choisir type de compte :** Compte local
- **Se connecter avec Microsoft :** Non

## Méthode alternative pour créer un compte local si l’option n’est pas disponible

1. Sur la page **"Se connecter avec Microsoft"** :
   - Appuyer sur les touches ``MAJ + F10``
   - Une **invite de commandes** s’ouvre, entrez la commande suivante :

```
control netconnections
```

   - **Désactiver** la carte réseau

2. Revenir sur la page de connexion via ``ALT + TAB``
   - Cliquer sur **Suivant**
   - Un message d’erreur apparaît, cliquez sur **Ignorer**
   - La page pour créer un **compte local** devrait maintenant apparaître

3. Réactiver la connexion Internet :
   - Revenir sur la page de configuration de la carte réseau via ``ALT + TAB``
   - **Réactiver** la carte réseau et fermer la fenêtre

4. Vous pouvez maintenant continuer l'installation de Windows normalement.
